# 悬浮球 Motion Lab

这是一个独立的悬浮球动画实验页，只验证 `ball / bar / panel` 的开合与收回观感，不接桌宠正式窗口逻辑。

## 运行方式

### 方式一：走项目根目录的 Vite

1. 在项目根目录执行：

```powershell
npm run dev
```

2. 打开浏览器访问：

```text
http://localhost:5173/demo/orb-motion-lab/
```

## 页面里能调什么

- 停靠边：左侧 / 右侧
- 策略：
  - 液体拉伸（单外壳连续 morph，球 -> 胶囊 -> 窗口）
  - 角落回球
  - 整边裁切
- 打开目标：Bar / Panel
- 动画时长
- 回球高度

## 建议怎么用

1. 先固定在右侧，并优先试“液体拉伸”。
2. 重点反复试 `ball -> panel -> ball`。
3. 如果某套方案主观上最顺，再把它迁回正式悬浮球。

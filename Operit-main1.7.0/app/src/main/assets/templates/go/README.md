# Operit Go 项目

这是一个使用 Operit 创建的 Go 项目。

## 快速开始

1. 点击 "go mod tidy" 整理依赖
2. 点击 "go run main.go" 运行程序
3. 点击 "go build" 构建可执行文件
4. 点击"浏览器预览"查看运行效果

## 项目结构

- `main.go` - 主程序入口
- `go.mod` - Go 模块定义
- `.operit/config.json` - Operit 工作区配置

## 自定义

您可以：
- 添加更多 Go 源文件和包
- 在 `go.mod` 中管理依赖
- 在 `.operit/config.json` 中自定义命令
